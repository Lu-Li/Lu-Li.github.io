Software Test Engineer at IBM

      My worked as a software engineer in IBM lasted for nearly 3 months. We are aiming at developing a Supply chain management software for China Mobile in this project. The team is made up of about 50 people. There are three parts of my tasks.

      At first, I was asked to help deploy the web server. I installed Tomcat and configured its environmental variables. After packaging wars using maven and building it using Hudson, I uploaded the project wars to Tomcat web server. Furthermore deployment of the wars was needed to make the whole project available through the domain index on the Internet.

      Then I was responsible for performing load test using LoadRunner. According to the Supply chain management workflow, the website is visited by staffs in different position around the country. In order to guarantee the performance of SCM software system under anticipated peak load conditions, I had to simulate multiple users accessing the program concurrently.

      Finally, I was in charged of doing manual UI tests and writing automation workflow test scripts using the Rational Functional Tester tool. This task enabled me to understand this supply workflow which coordinate upstream and downstream flows of products, services, finances, or information from a source to a customer.
